\chapter{Data Archive Contents}
\label{appendix}

The data archive accompanying this work is a copy of the \cc{tcoof-trust} GitHub
repository. Its up-to-date version can always be found at the following URL:
\href{https://github.com/matejcik/tcoof-trust}{\ttfamily https://github.com/matejcik/tcoof-trust}

The code uses the Scala Build Tool (SBT) for building, running, and generating
documentation. Please refer to \cc{README.md} for details.

To run the example code, use \cc{sbt run}.

To build the API documentation in HTML format, use \cc{sbt doc}. The resulting
documentation will be stored in \cc{target/scala-2.12/api}.

To run the unit test suite, use \cc{sbt test}.

\begin{filepath}

\begin{dsldesc}

    \cc{resultlib} is a small library of common functions.
\end{dsldesc}
